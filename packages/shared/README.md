# @deckgl-fiber-renderer/shared

> **Internal utilities package.** Not intended for direct use.

Shared utilities used across `@deckgl-fiber-renderer/*` packages:

- **Logging** - loglayer wrapper for development mode logging
- **Store** - zustand store for deck.gl instance management
- **Utilities** - Common functions (isDefined, isFn, noop, toPascal, isBrowserEnvironment)

## For Users

Install [@deckgl-fiber-renderer/dom](../dom) instead. This package is a dependency and will be installed automatically.
